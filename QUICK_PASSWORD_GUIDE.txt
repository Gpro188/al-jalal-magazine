=====================================
CLASS UNION LOGIN & PASSWORD EXPLAINED
=====================================

❗ IMPORTANT: Class Unions DON'T have shared passwords!

HOW IT WORKS:
-------------
Each STUDENT has their OWN individual account:
- Personal email
- Personal password  
- Assigned to specific union

EXAMPLE:
--------
Alpha Union has 3 students, each with DIFFERENT logins:

Student 1:
  Email: ahmed@student.edu
  Password: Ahmed123!
  Union: Alpha Union

Student 2:
  Email: mohammed@student.edu
  Password: Mohammed123!
  Union: Alpha Union

Student 3:
  Email: fatima@student.edu
  Password: Fatima123!
  Union: Alpha Union

All in SAME union, but DIFFERENT passwords!

HOW STUDENTS LOGIN:
-------------------
1. Go to union portal: http://localhost:3002/login/alpha
2. Enter THEIR OWN credentials:
   - Email: ahmed@student.edu
   - Password: Ahmed123!
3. Click "Sign In"
4. Can submit articles to Alpha Union

NOT LIKE THIS:
--------------
❌ WRONG: One password for all Alpha Union students
❌ Union: Alpha Union
❌ Shared Password: AlphaPass123 (everyone uses this)

CORRECT WAY:
------------
✅ Each student has UNIQUE email and password
✅ All assigned to same union in Firestore
✅ Individual accountability and tracking

ADMIN CREATES ACCOUNTS:
-----------------------
Step 1: Firebase Authentication
  - Add user: ahmed@student.edu
  - Password: Ahmed123!
  - Copy UID

Step 2: Firestore → users collection
  Document ID: [paste UID]
  Fields:
    - email: ahmed@student.edu
    - role: contributor
    - displayName: Ahmed Ali
    - classUnion: Alpha Union

Step 3: Give to student
  Login URL: http://localhost:3002/login/alpha
  Email: ahmed@student.edu
  Password: Ahmed123!

ALL USER TYPES:
---------------
Type          | Own Password? | Union?     | Login URL
--------------|---------------|------------|------------------
Student       | ✅ Yes        | ✅ Yes     | /login/alpha
Editor        | ✅ Yes        | ⚪ Assigned| /login
Admin         | ✅ Yes        | ⚪ All      | /login

KEY POINTS:
-----------
✅ No shared union passwords
✅ Every user has individual login
✅ Union portals are branded (logo/colors)
✅ Students auto-assigned to union
✅ Personal submission history
✅ Secure Firebase Authentication
