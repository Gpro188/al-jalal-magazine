# 🚀 Quick Start Guide - AL-JALAL Magazine

## For School Administrators

### Step 1: Access the System
- **URL**: http://localhost:3000 (development) or your deployed URL
- **Admin Login**: Use `/login` → Click "Editor/Staff Login"

### Step 2: Set Up Class Unions
1. Go to **Admin Dashboard** → **Manage Class Unions**
2. Create unions (e.g., Alpha, Beta, Gamma, Delta)
3. Add logos or emojis for each union
4. Choose theme colors

### Step 3: Create Editor Accounts
1. Go to **Admin Dashboard** → **Manage Editors**
2. Fill in editor details:
   - Email
   - Password (temporary)
   - Display Name
   - Assign to unions
3. Share credentials with editors

### Step 4: Create Student Accounts
**Option A - Manual Creation:**
- Admin creates accounts one by one
- Best for small groups

**Option B - Bulk Creation:**
- Collect student information
- Create accounts in batches
- Distribute credentials through union reps

**Required Info for Each Student:**
- Full Name
- Email Address  
- Class Union
- Temporary Password

### Step 5: Launch!
- Announce to students
- Share login instructions
- Start accepting submissions

## For Students

### Your First Time:
1. Visit the login page
2. Click on your Class Union
3. Enter your email and password
4. Start writing!

### Submit an Article:
1. Login to your dashboard
2. Click "Submit New Article"
3. Fill in:
   - Title
   - Category (Literature, Science, Arts, etc.)
   - Content
   - Your name
4. Click "Submit"
5. Wait for editor review

### Check Status:
- **Pending Review**: Editor hasn't reviewed yet
- **Staff Approved**: Editor approved, waiting for admin
- **Published**: Live in the magazine!
- **Rejected**: See feedback and resubmit

---

## For Editors

### Review Submissions:
1. Login to Editor Dashboard
2. See pending articles from your unions
3. Read each submission
4. Click "Approve" or "Reject"
5. Add feedback if rejecting

### Tips for Good Feedback:
- Be constructive and encouraging
- Point out specific improvements
- Celebrate good writing
- Help students learn

## 🆘 Troubleshooting

**Problem**: Student can't login  
**Solution**: Check email spelling, reset password, verify account exists

**Problem**: Editor not seeing submissions  
**Solution**: Verify editor is assigned to correct unions

**Problem**: Can't publish articles  
**Solution**: Ensure article is "Staff Approved" first

**Problem**: Union not showing on login page  
**Solution**: Refresh page, check if union exists in database
